# Requirements Archive: v1.0 MVP

**Archived:** 2026-03-05
**Status:** All 34 v1 requirements complete
**Core Value:** Every @chillwhales/* package installs cleanly, works in any JS/TS environment (ESM and CJS), and ships with correct types — consumers never think about compatibility.

## v1 Requirements (All Complete)

### Build & Shared Configs

- [x] **BUILD-01**: All packages share a single build configuration source, eliminating duplicate build.config.ts files — *Validated: @chillwhales/config createBuildConfig() factory*
- [x] **BUILD-02**: All packages share a single test configuration source, eliminating duplicate vitest.config.ts files — *Validated: @chillwhales/config createVitestConfig() factory*
- [x] **BUILD-03**: Build process fails on warnings (failOnWarn: true) to surface hidden issues — *Validated: Default in createBuildConfig()*
- [x] **BUILD-04**: Dependency versions are centralized via pnpm catalogs in pnpm-workspace.yaml — *Validated: 25 entries in default catalog*
- [x] **BUILD-05**: Circular dependencies between packages are detected and reported — *Validated: madge in CI*

### Code Quality

- [x] **QUAL-01**: All packages are linted by Biome from a single root configuration — *Validated: biome.json at root*
- [x] **QUAL-02**: All packages are formatted by Biome from a single root configuration — *Validated: 92 files formatted*
- [x] **QUAL-03**: Pre-commit hook runs biome check before every commit — *Validated: simple-git-hooks pre-commit*
- [x] **QUAL-04**: Unused dependencies and exports are detected via knip — *Validated: knip.json with workspace config*
- [x] **QUAL-05**: Monorepo package consistency is validated via sherif — *Validated: sherif with failOnWarnings*
- [x] **QUAL-06**: Only pnpm can be used as package manager (enforced via preinstall) — *Validated: npx only-allow pnpm*
- [x] **QUAL-07**: All commits follow conventional commit format, enforced via commitlint — *Validated: commitlint.config.mjs + commit-msg hook*

### CI Pipeline

- [x] **CI-01**: Every PR runs install, typecheck, lint, format check, build, and test — *Validated: 4-layer CI pipeline, 11 jobs*
- [x] **CI-02**: In-progress CI runs are cancelled when new commits are pushed — *Validated: concurrency + cancel-in-progress*
- [x] **CI-03**: Package exports and types are validated via publint and attw before merge — *Validated: CI pkg-verify job*
- [x] **CI-04**: Test coverage is reported as a PR comment on every PR — *Validated: vitest-coverage-report-action*

### Release Automation

- [x] **REL-01**: Changesets is configured for independent versioning of all packages — *Validated: .changeset/config.json*
- [x] **REL-02**: Changelogs are auto-generated per package with GitHub PR attribution — *Validated: @changesets/changelog-github*
- [x] **REL-03**: Scoped packages are configured for public npm access — *Validated: access: public*
- [x] **REL-04**: After a PR merges to main, a version PR is automatically created or updated — *Validated: release.yml changesets/action*
- [x] **REL-05**: When the version PR merges, changed packages are automatically published to npm — *Validated: release.yml publish step*
- [x] **REL-06**: PR-based snapshot releases are available via pkg-pr-new — *Validated: preview.yml*
- [x] **REL-07**: GitHub Releases are created when packages are published — *Validated: contents:write permission*

### Package Metadata

- [x] **PKG-01**: LICENSE file exists at root and is included in each published package — *Validated: 15 packages/*/LICENSE*
- [x] **PKG-02**: Each package has a README with description, install command, and basic usage example — *Validated: 10/15 packages (5 newer packages created after Phase 8 lack READMEs)*
- [x] **PKG-03**: Each package.json has complete fields: files, engines, repository, keywords — *Validated: All packages*
- [x] **PKG-04**: All viem-using packages correctly declare viem as a peer dependency — *Validated: 6 viem-using packages*

### Testing & Coverage

- [x] **TEST-01**: Test coverage is measured via @vitest/coverage-v8 — *Validated: v8 provider configured*
- [x] **TEST-02**: Minimum coverage thresholds are enforced (70-80%) — *Validated: 80% thresholds, 94%+ baseline*

### Rename & LICENSE (Phase 07.1 — Inserted)

- [x] **RENAME-01**: lsp30 package fully renamed to lsp31 — *Validated: packages/lsp31 exists, lsp30 removed*
- [x] **LICENSE-01**: LICENSE file permanently exists in each publishable package directory — *Validated: 15 committed LICENSE files*
- [x] **LICENSE-02**: prepack/postpack LICENSE copy scripts removed from all package.json files — *Validated: Zero matches*

### External Extraction

- [x] **EXT-01**: Reusable utilities from chillwhales/marketplace are identified and extracted — *Validated: 15 utils modules, LSP enrichments*
- [x] **EXT-02**: Reusable utilities from chillwhales/lsp-indexer are identified and extracted — *Validated: lsp1 + erc725 packages*
